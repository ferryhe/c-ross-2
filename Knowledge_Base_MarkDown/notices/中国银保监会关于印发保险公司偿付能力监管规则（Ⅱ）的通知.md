# 中国银保监会关于印发保险公司偿付能力监管规则（Ⅱ）的通知


发布日期：
2021-12-30 20:30

来源：
财会部(偿付能力部)

原始链接：
https://www.nfra.gov.cn/cn/view/pages/governmentDetail.html?docId=1027892&itemId=861&generaltype=1

各银保监局，各保险集团（控股）公司、保险公司、保险资产管理公司：

为改进和加强保险公司偿付能力监管，增强保险业服务实体经济能力，有效防范保险市场风险，更好地保护保险消费者利益，银保监会制定了《保险公司偿付能力监管规则（Ⅱ）》，已于2021年9月9日经2021年第12次委务会议审议通过，现予印发。
 
中国银保监会
 
 2021年12月30日
