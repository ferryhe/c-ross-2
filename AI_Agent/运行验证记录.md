# 运行验证记录

日期：2026-04-03

## 目的

验证当前分支上的“数据层 + RAG 层 + chatbot 验证层”是否能够在本地装依赖后跑通。

## 已安装依赖

Python：

- 按 `AI_Agent/requirements.txt` 安装
- 关键包包括：
  - `openai`
  - `faiss-cpu`
  - `numpy`
  - `tiktoken`
  - `python-dotenv`
  - `pytest`
  - `fastapi`
  - `uvicorn`
  - `httpx`

Frontend：

- 在 `AI_Agent/frontend/` 下执行 `npm ci`

## 实际执行过的验证命令

### Python

```powershell
python -m pip install -r AI_Agent\requirements.txt
python -m pytest tests AI_Agent\tests
python -X utf8 AI_Agent\scripts\evaluate_regulatory_answers.py --model gpt-5.4-mini --compare-agentic
```

结果：

- `72 passed`
- 生成真实问答报告：`AI_Agent/eval/gpt-5.4-mini-20260403T190524Z.json`

### Frontend

```powershell
cd AI_Agent\frontend
npm.cmd ci
npm.cmd test
npm.cmd run build
```

结果：

- `7 passed`
- `vite build` 成功

## 备注

- PowerShell 默认策略会阻止 `npm.ps1`，因此本机验证时使用了 `npm.cmd`。
- 当前仓库包含预构建的 `frontend/dist/`，因此本次构建后的产物也一并更新。
- 真实问答结论见：`AI_Agent/真实问答验证记录-gpt-5.4-mini.md`
